## [1 min] 准备环境

安装 Python、Django。

### Linux and OS X
Python 已经默认安装，只需要先安装 pip（是什么？）：

  curl https://raw.github.com/pypa/pip/master/contrib/get-pip.py | python

然后安装 Django：

	pip install Django

以上。好简单！

### Windows
下载 Python 并安装（注意要下载 2.x 的版本）：

	http://python.org/getit/
